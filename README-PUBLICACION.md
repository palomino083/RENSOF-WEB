# Publicacion de rensof.pe

Este paquete contiene una version estatica lista para subir al hosting del dominio rensof.pe.

## Formulario

El formulario envia a contacto@rensof.pe mediante FormSubmit. La primera vez que se envie, FormSubmit puede pedir confirmacion del correo receptor. Si el correo oficial es otro, cambia esa direccion en contacto.html.

## Subida al hosting

1. Ingresa al panel de hosting o cPanel de rensof.pe.
2. Abre la carpeta publica del dominio, normalmente public_html.
3. Sube el contenido de esta carpeta, no la carpeta completa.
4. Verifica que index.html quede directamente dentro de public_html.
5. Abre https://rensof.pe y revisa navegacion, formulario y vista movil.

No hice la publicacion directa porque se necesitan accesos del hosting, FTP, cPanel o repositorio conectado al dominio.
