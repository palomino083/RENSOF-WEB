# RENSOF Design System 2030

Base visual y de producto para posicionar a RENSOF como plataforma de inteligencia estrategica.

## 1) Posicionamiento

- Categoria: Plataforma de Inteligencia Estrategica
- Slogan: Transformando datos en decisiones estrategicas.
- Tono: Claro, sobrio, tecnologico, orientado a resultados.

## 2) Principios de interfaz

- Menos texto, mas evidencia visual.
- Datos accionables antes que descripcion.
- Navegacion por historias, no por catalogo.
- Consistencia visual en todos los productos REN.
